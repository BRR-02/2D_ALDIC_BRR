# 2D_ALDIC_v1
AL-DIC(Augmented Lagrangian DIC) is a fast, parallel-computing DIC algorithm, which combines advantages of Local Subset DIC (fast, compute in parallel) and Global DIC (guarantee kinematic compatibility).
